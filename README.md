# 🎓 AI-Enabled Student Admission System (using Salesforce Agentforce)

An AI-powered Salesforce CRM application that streamlines student admissions through automation, intelligent decision-making, and secure data management. The system leverages Salesforce Flow, Approval Processes, Reports, Dashboards, and Agentforce AI to improve admission efficiency and enhance the student support experience.

- Team ID:	SWTID-2026-9246
- Developed by: APPALA GAYATRI AKHILA

---

## 📌 Project Overview

Traditional admission systems often rely on manual processes and spreadsheets, leading to duplicate records, delayed approvals, and inefficient communication. This project provides a centralized Salesforce solution that automates the complete admission lifecycle while incorporating AI-powered insights for faster and smarter decision-making.

The application enables admission counselors and administrators to manage student records, courses, enrollments, approvals, and notifications through an intuitive Lightning interface.
